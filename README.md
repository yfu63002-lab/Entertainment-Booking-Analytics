# Entertainment Booking Analytics

## Overview
This project analyzes an entertainment booking database to generate business insights related to agent performance, customer behavior, entertainer profitability, and seasonality. The project was completed as part of an MSBA coursework and demonstrates SQL-driven analytics supported by Python-based data validation and visualization.

---

## What This Project Shows
- Ability to work with a **relational database** and multi-table joins
- Strong **SQL skills** for data quality checks and analytical queries
- Use of **Python (pandas, matplotlib)** for data cleaning and exploratory analysis
- Translation of data results into **business insights and recommendations**
